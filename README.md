# Nextjs tailwindcss starter

## Install deps

Installing the dependencies

```bash
yarn
```

## Debug locally

```bash
yarn dev
```

## Build

```bash
yarn build
```

## Built with

- Next.js
- Typescript
- Tailwindcss
